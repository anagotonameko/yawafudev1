# やわふで / Yawafude

「やわふで」は、やわらかく・細く・力を抜いた手書き風フォントです。  
英字・ひらがな・カタカナ・数字・一部記号に対応しています。

ノート、日記、ゲームUI、同人作品など、  
「主張しすぎない文字」が欲しい場面で使いやすいフォントを目指しました。

---

## 対応文字

- 英字（A–Z, a–z）
- 数字（0–9）
- ひらがな
- カタカナ
- 一部記号（! ? () [] {} など）

※ すべての文字を網羅しているわけではありません。

---

## 使用例

- メモ・日記
- ゲーム内テキスト
- 同人誌・ZINE
- Webや画像作品の英字・かな表記

---

## 使用条件・ライセンス

- 個人・商用利用：OK
- 改変：OK
- 再配布：OK（フォント単体での再配布は不可）
- クレジット表記：不要（逆に書かれると恥ずかしいからやめて欲しい）

※ライセンスとかはよく分かんないのでこれだけです。明らかに犯罪なことはやめてね。
---

## 注意事項

	⁃	手書き風フォントのため、長文や小さいサイズでは読みにくくなる場合があります。マジで読みにくくなります。
	⁃	文字がすんごい変かもしれません。ごめんね。無料フォントだから許してください。
	⁃   本フォントを使用したことによるトラブルについて、制作者は責任を負いません。
